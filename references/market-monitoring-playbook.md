# Market Monitoring Playbook

## Default execution order

### Step 1 — establish scope
Decide whether the task is:
- holdings-first review
- full market sweep
- one-symbol follow-up
- event response scan
- formatting only

### Step 2 — collect market structure
Get enough information to know:
- what is moving
- how broad the move is
- whether the move is confirmed by activity

### Step 3 — collect event context
Get the macro and crypto-native events that could explain the move.

### Step 4 — map events to holdings
If the user has holdings, identify:
- direct exposure
- correlated exposure
- symbols that need immediate review

### Step 5 — filter attention noise
Use social heat only as support.
It should not dominate the result unless confirmed elsewhere.

### Step 6 — rank output
Produce:
1. top holdings risks
2. top watchlist names
3. top uncertain but noteworthy names

### Step 7 — summarize cleanly
Good final outputs are short, ranked, and honest about uncertainty.
