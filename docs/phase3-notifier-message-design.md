# Phase3 Notifier Message Design

## 目标

通知必须优先回答：

1. 现在风险高不高
2. 是否需要叫醒 Hermes
3. 当前最值得关注的 OKX 可交易品种有哪些

## 推荐区块顺序

1. 运行状态
2. 持仓 / 方向偏置 / 风险等级
3. `Trigger 判定`
4. `热度排名`
5. `API Health`
6. `持仓风险`
7. `热门可交易品种`
8. `宏观 / 地缘变化`
9. `重点新闻`

## 热度排名区块

建议拆分展示：

- 当前列表
- 持仓优先
- 白名单热议
- CMC 补充
- OKX涨幅榜
- OKX持仓异动
- OKX高持仓

## 低变化场景

若本轮无重要变化，应压缩成短报：

- 持仓
- 方向/风险
- 观察级触发
- 热度排名

## 不应再出现的内容

- 旧版优先币种命名
- 任何自动执行/下单措辞
- 把热榜表述成交易指令
