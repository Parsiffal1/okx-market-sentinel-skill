# Telegram Brief Template

Use this when the user wants a compact, high-signal message.

## Rules
- Keep it short.
- Keep the ranking.
- Keep the main driver.
- Keep one uncertainty line.
- If holdings exist, list them before new watchlist names.

```text
OKX Market Sentinel | Scan complete

Market tone
- Bias:
- Escalation:
- Main driver:

Priority holdings
1. [symbol] — [why it matters now]
2. [symbol] — [why it matters now]

Symbols worth monitoring
1. [symbol] — [reason]
2. [symbol] — [reason]
3. [symbol] — [reason]

Confidence notes
- [what is well confirmed]
- [what is still uncertain]
```
