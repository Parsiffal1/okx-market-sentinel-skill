# Example Use Cases

## Use case 1 — operator morning scan
An operator wants a fast answer to: `what do I need to care about before the next session opens?`

## Use case 2 — holdings review before action
A trader or risk manager wants a holdings-first explanation before making any portfolio change.

## Use case 3 — event response
A major macro or crypto-native headline hits and the user wants a fast, grounded interpretation.

## Use case 4 — downstream agent input
Another agent needs a compact market state summary to feed into a broader research or reporting workflow.
