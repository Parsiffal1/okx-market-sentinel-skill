# Telegram Brief Template

Use this when the user wants a compact, high-signal message.

## Rules
- Keep it short.
- Keep the ranking.
- Keep the main driver.
- Keep one uncertainty line.
- If holdings exist, list them before new watchlist names.
- Preserve the key schema labels so another agent can reuse the brief.

Market Sentinel | Scan complete

- regime_classification: 
- action_state: 
- escalation_needed:

- [driver 2]
